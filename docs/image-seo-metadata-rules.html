四、各字段写作规则（Prompt 约束）

alt_text_en — 客观短描述
只描述看得见的内容；适合读屏；不夸张、不营销。例：男人 + 白色淋浴椅 + 扶手 + 浴室。
caption_en — 带卖点的图注
比 alt 稍长；可强调舒适、宽座、与竞品对比等卖点。缺省时自动回填 alt_text_en。
image_description_en — 最完整叙述
可包含对比图信息、手持花洒等细节；写入 XMP-dc:Description。
tags_en — 检索关键词
数组，最多约 8 个；覆盖产品类型、场景、人群、功能。
new_file_name — SEO 文件名
小写英文、连字符、无扩展名、最多 12 词。例：man-using-shower-chair-with-armrests
brand / model — 用户注入
不由 AI 生成；表单传入后原样写入 IPTC:Credit / EXIF:Model。

全局约束

规则说明
只写可见内容禁止编造品牌、材质、型号、功能、场景
双语生成，英文写入_en 写入图片；_zh 仅供 UI 审核
置信度confidence_note 仅 certain / uncertain，不写入图片
缺省回填caption 空 → 用 alt；description 空 → 用 caption
